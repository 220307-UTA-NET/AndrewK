Tables

Tables are a common way to give an html page structure as well!

  • The <table> tag - creates the table
  • The <caption> tag - to list a caption for the table
  • The <thead> tag - to specify the content of a header row
  • The <tr> tag - to specify the new row
  • The <th> tag - to specify the content of the header cell
  • The <tbody> tag - to specify the content of the body of the table
  • The <td> tag - to specify the content of a cell in the body
  • The <tfoot> tag - to specify the content of the footer row
